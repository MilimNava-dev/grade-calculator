# Grade Tracker

This is a web application for tracking and calculating student grades, built with React and Vite. It features a modern and responsive UI using Tailwind CSS and shadcn/ui.

## ‚ú® Features

- **Subject Management**: Create, view, and delete subjects.
- **Grade Tracking**: Add, edit, and delete grades for each subject.
- **Weighted Categories**: Assign different weights to grade categories for accurate score calculation.
- **Automatic Calculation**: Automatically calculates the final score for each subject based on grades and weights.
- **Local Persistence**: All data is saved in the browser's local storage, so your information is available across sessions.

## üöÄ Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/MilimNava-dev/grade-calculator.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd grade-calculator
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

### Running the Application

To start the development server, run:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

## Available Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the application for production.
- `npm run lint`: Lints the code using ESLint to ensure code quality.
